## Express

```bash
yarn add express
```

## Dotenv

```bash
yarn add --dev dotenv
```

## Typescript

```bash
yarn add --dev typescript @types/express @types/node
npx tsc --init
```

## TypeORM

```bash
yarn add typeorm
npx typeorm init --name ToDoList --database sqlite
```
