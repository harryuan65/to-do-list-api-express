# To-do List Server - Express

This is a simple to-do list api server using Express & Typescript.

## Environment

| env        | version  |
| ---------- | -------- |
| node       | v16.15.1 |
| yarn       | 1.22.19  |
| express    | 4.18.2   |
| typescript | 4.9.4    |

## Development

```bash
yarn install
yarn dev
```

Server should be running on PORT 3004 by default(Or you can create a `.env` file to set the `PORT` variable).
